# Arcadium

A command line arcade games written in python to pass time.


## Table of contents

* [Installation]()

## Installation
To install **Arcadium** use these commands : 
```
pip install inquirer
git clone https://github.com/HushmKun/Arcadium/
cd Arcadium 
python main.py
```
